# Tip of the Week

Every week a team member of Digital Produt & Tech will present something this person learned that could be useful to peers. Topics can be new tools or software, useful CLI commands, shortcuts, etc.

## Rules

<Steps>
### Max 25 minutes meeting

- Max 10 minutes presentation.
- 10 minutes for questions and discussion.
- 5 minutes for setup and time buffer.

### The tip should be presented in a short and concise way

- What is the Tip?
- Why is it useful?
- How to use it?
- Example of use and where to find more information.

### The presentation need to be uploaded to the repository in the folder [`presentations`](pages/presentations)

- The file should be named `YYMMDD-<topic>.mdx`.
- Use [000000-template.mdx](pages/presentations/000000-template.mdx) as a template.
- In case you need to add images or other files, create a folder with the same name as the file and add the files there.

</Steps>
